# RoboIndex 版本更新日志

---

## v0.0.2 — 2026-04-10

**线上地址：** https://roboindex.github.io/
**对比基准：** v0.0.1（2026-03-12，数据初始化阶段）

### 一、数据规模扩张

| 指标 | v0.0.1 | v0.0.2 | 增幅 |
|------|--------|--------|------|
| 收录论文总数 | 26 篇 | **627 篇** | +2311% |
| 含开源代码论文 | — | **126 篇** | — |
| 数据来源 | 手动录入 | GitHub API 自动发现 | — |

- 新增自动化论文发现脚本（`scripts/fetch-papers.ts`），基于 GitHub API 搜索 23 组关键词，覆盖 IEEE RA-L 2020–2025 年的开源论文
- GitHub Actions 每日自动同步，数据自我维护无需人工干预

### 二、新增页面

**论文列表页（/papers）**
- 支持按年份、标签筛选，含关键词搜索
- 每篇论文展示：标题、作者、年份、摘要、开源代码链接

**论文详情页（/papers/[slug]）**
- 结构化展示论文完整信息：摘要、标签、GitHub 仓库、引用格式

**投稿指南页（/guides）**
- 完整收录 Ce Hao 撰写的 IEEE RA-L 投稿攻略，内容零损失
- 三阶段结构（初稿 → 审稿 → 接收），Reddit 风格可折叠目录
- 侧边栏固定导航，长文阅读体验优化

**数据仪表盘（/dashboard）**
- 论文数量年度趋势图
- 研究主题标签分布 Top 20
- 审稿周期时间线（竖向节点 + 耗时 pills）
- 全站数据以橙色主题风格统一

**ral.skill 介绍页（/ral-skill）**
- 展示 AI 投稿助手的 7 大功能模块
- 真实 Demo 对话示例（投稿流程、催审建议、格式检查等）
- 三大知识来源说明（IEEE 官网 · 投稿攻略 · 社区经验）

**Showcase 开源论文展示页（/showcase）**
- 专门展示有开源代码的论文，支持标签 / 年份筛选
- 接入 GitHub API 展示各仓库 Star 数、语言、最后更新时间

**Contribute 贡献页（/contribute）**
- 说明数据提交方式（PR / Issue）
- 介绍核心团队，展示项目愿景

### 三、首页视觉升级

**动态世界地图背景**
- 纯 SVG 描边世界地图，无图片依赖，加载极快
- 标注 8 个机器人领域顶级机构城市（MIT · CMU · ETH · 帝国理工 · 清华 · 东大 · 新国大 · 伯克利），每个节点有持续脉冲扩散动画
- 地图整体有 8s 周期呼吸渐变，深色模式下自动调整透明度

**毛玻璃蒙版效果**
- 地图（z=0）→ 椭圆毛玻璃蒙版（z=5）→ 文字（z=10）三层结构
- backdrop-filter blur(60px) 强毛玻璃，中心向边缘渐变消散，文字全程清晰锐利
- 深色模式专属配色，完整适配

**访客实时位置光点**
- 根据访问者 IP 自动定位，在地图对应经纬度显示蓝色闪烁光点
- 双环脉冲动画（1.6s），比城市节点动画更快更亮以示区分

### 四、访客数据统计

- 接入 GoatCounter（无 Cookie、GDPR 友好）隐私合规统计工具
- 首页底部实时显示累计访客数
- 同源缓存方案：GitHub Actions 每小时将统计数据同步至 `public/visitor-count.json`，前端优先读取同源文件，绕过广告拦截器，保证数据可见

### 五、深色模式

- 全站支持深色模式，Header 右上角手动切换按钮
- CSS 变量体系（`--surface-0/1/2`、`--text-primary/secondary/muted`、`--border`）全局统一，切换无闪烁

### 六、移动端适配

- 全站 7 个页面完成响应式修复（首页、/papers、/papers/[slug]、/guides、/ral-skill、/showcase、/dashboard）
- 毛玻璃遮罩从固定 860px 改为 `min(860px, 95vw)` 视口自适应
- 标题字号全部增加响应式断点（`text-3xl sm:text-4xl md:text-5xl`）
- 内边距统一为 `px-4 sm:px-6`
- ral.skill 页三列卡片在手机端自动降级为单列

### 七、工程基础设施

- Next.js 16 App Router，静态导出（`output: 'export'`），托管 GitHub Pages，零服务器成本
- GitHub Actions 自动化：论文数据每日更新、访客计数每小时同步、代码推送即触发部署
- 深色模式基于 CSS class，无 SSR hydration 问题
